Overview

React application that renders and displays terrain layers collected by NASA in an interactive way. Browse and explore terrain layers of the Moon, Mars and Vesta. Learn about how NASA collects data through interactive layers of weather, heat signals, and terrain depth. Pinpoints significant landmarks on the Moon, Mars and Vesta with geological information. Pulls daily articles written by NASA to display in news feed.

Visit Project View Code

Technologies Used:

  • React
  • React Router
  • Redux
  • NASA's Trek API
  • Esri API
  • Sass

Wow and Smooth Scroll were implemented into the project to bring Jquery like features without the library's heavy footprint.
